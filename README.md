# CivicEase Chatbot

CivicEase Chatbot is a Flask-based chatbot designed to assist citizens by answering common civic issues related to Nashik Municipal Corporation (NMC). It provides quick information on topics like complaint filing, tracking, department contacts, emergency services, and municipal announcements.

This chatbot is planned to be integrated with the CivicEase app, which is used by citizens to raise and track civic issues efficiently.

## Features

- Conversational interface for common civic queries.
- Abusive language filtering for polite interactions.
- Department-wise guidance (water, electricity, roads, illegal construction, etc.).
- Emergency contact info (police, ambulance, fire).
- Instructions for complaint filing and tracking.
- Links to official resources and apps for extended services.
